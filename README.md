## Para testar em sua maquina

```bash

# Clone no seu repositório
$ git clone git@github.com:lipecalixto/todo-list.git

# Acesse a o diretório
$ cd todo-list

# Instale às Dependências
$ npm install

# Rode a aplicação
$ npm run dev

```
